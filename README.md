# APRICOT19

## How to use

### Pre-requirements
- git
- python3

### Linux
```bash
git clone https://github.com/munhyunsu/APRICOT19.git
cd APRICOT19
chmod +x 0-zero-day.sh
./0-zero-day.sh
python3 start.py
```
